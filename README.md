# 🚀 Simple Order Management Frontend

Frontend project using **Nuxt 3** + **Tailwind CSS v4**.

## 📌 Features

- ✅ Nuxt 3 (Vue 3)
- ✅ Tailwind CSS v4 with PostCSS
- ✅ Fetching API for order management

## 🚀 Setup

1️⃣ **Install dependencies**

```sh
npm install
```

#### 2️⃣ **Run the project**

```sh
npm run dev
```
